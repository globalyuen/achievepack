# 产品轮播图显示验证指南

## ⚠️ 重要提示
产品轮播图**只在 pouch.eco 域名**上显示，不在 achievepack.com 上显示！

## 验证步骤

### 1. 确认访问的域名
请确保您访问的是：
- ✅ 正确：https://pouch.eco/
- ❌ 错误：https://achievepack.com/

### 2. 清除浏览器缓存
如果您之前访问过网站，需要清除缓存：

**Chrome/Edge:**
- Mac: Cmd + Shift + Delete
- Windows: Ctrl + Shift + Delete
- 选择"缓存的图片和文件"
- 点击"清除数据"

**Safari:**
- Safari菜单 → 偏好设置 → 高级
- 勾选"在菜单栏中显示开发菜单"
- 开发菜单 → 清空缓存

### 3. 强制刷新页面
- Mac: Cmd + Shift + R
- Windows: Ctrl + Shift + R

### 4. 轮播图位置
轮播图位于首页的以下位置：
```
POUCH.ECO 首页
├── 导航栏
├── Hero 区域（"Your Brand. Our Eco Packaging."）
├── 【★ 产品照片轮播图 ★】← 应该在这里！
│   标题："See Our Work"
│   副标题："REAL PRODUCTS • REAL BRANDS"
│   ↓
│   [大型轮播图展示 25 张产品照片]
│   ↓
│   [左右箭头按钮]
│   ↓
│   [指示器点（25个点）]
│   ↓
│   [缩略图预览（桌面端）]
├── Features Grid（"Why Choose Pouch.eco?"）
└── ...
```

### 5. 视觉特征
轮播图应该具有以下特征：
- 🖼️ 粗黑色边框（4px）
- 💛 黄绿色悬停效果（#D4FF00）
- 💚 绿色活动状态（#10b981）
- ◀️ 左箭头按钮（白色方框）
- ▶️ 右箭头按钮（白色方框）
- ⏺️ 25个指示器点
- 🖼️ 底部缩略图（仅桌面端）
- 📊 右上角计数器："1 / 25"

## 仍然看不到？

### 方案 1：使用无痕模式
打开浏览器的无痕/隐私模式窗口，直接访问：
https://pouch.eco/

### 方案 2：检查开发者控制台
1. 按F12打开开发者工具
2. 切换到Console标签
3. 查找任何JavaScript错误
4. 截图发给开发团队

### 方案 3：等待缓存失效
Vercel的边缘缓存可能需要最多5-10分钟才能完全更新。
